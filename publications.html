Websites/ Digital projects

October 2018
Website: Speaking Characters in Extant Classical Greek Tragedies
This site displays a list of extant classical Greek tragedies and the speaking characters in those plays.

P.S. Given that this website on Greek characters is hosted on a free server, it may often not work properly. The advanced user can download the source code and run it from his/her own computer. This website is looking for an academic institutional host who can provide free server space. Contact me if you would kindly agree to host this website on Greek characters.

This was perhaps the first project in India to use XML-TEI.
